## 우선순위 권장사항

### 높은 우선순위 (사용자에게 자주 노출)
1. **AI 챗봇 로딩 메시지** (`'관련 자료를 검색하는 중...'`) - 사용자가 채팅 입력 시 가장 자주 보는 메시지
2. **로딩 메시지 상수 파일** (`loadingMessages.ts`) - 모든 카테고리별 로딩 메시지
3. **AI 챗봇 에러 메시지** - 사용자 경험에 직접적인 영향
4. **복습 캐러셀 UI 라벨** - 복습 기능 사용 시 자주 노출

### 중간 우선순위
5. **인증 관련 UI 텍스트** - 로그인/회원가입 시에만 노출
6. **채팅 사이드바 라벨** - 채팅 기록 확인 시 노출

### 낮은 우선순위 (개발자용 또는 드물게 노출)
7. **공유 라이브러리 에러 메시지** - 에러 발생 시에만 노출
8. **폼 검증 메시지** - 입력 오류 시에만 노출

---

## 참고사항

- 현재 i18n 시스템은 `next-intl`을 사용하고 있으며, 번역 파일은 `src/shared/i18n/messages/ko.json`과 `en.json`에 있습니다.
- 일부 컴포넌트는 이미 `useTranslations` 훅을 사용하고 있지만, 위에 나열된 텍스트들은 아직 하드코딩되어 있습니다.
- 로딩 메시지의 경우 `loadingMessages.ts` 파일을 i18n으로 전환하거나, 각 메시지를 번역 파일로 이동해야 합니다.
